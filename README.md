# IRSR-Infrared-Image-Super-Resolution-Reconstruction-
Reproduction of several infrared image super-resolution reconstruction algorithms, including deep learning-based methods and traditional algorithms.
K. Choi, C. Kim, M.-H. Kang, and J. B. Ra, "Resolution improvement of infrared images using visible image information," IEEE Signal Process. Lett., vol. 18, no. 10, pp. 611-614, Oct. 2011.
